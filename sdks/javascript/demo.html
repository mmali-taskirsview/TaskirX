📚 Integration Guide

Step 1: Include the SDK

<script src="adx-sdk.js"></script>

Step 2: Initialize

AdxSDK.init({ publisherId: 'YOUR_PUBLISHER_ID', apiUrl: 'https://your-adx-domain.com/api', // optional debug: true // optional, for development });

Step 3: Display Ads

// Banner Ad AdxSDK.showBanner({ containerId: 'ad-slot-1', size: '300x250' }); // Native Ad AdxSDK.showNative({ containerId: 'ad-slot-2' }); // Video Ad AdxSDK.showVideo({ containerId: 'ad-slot-3', size: '640x480' });

Step 4: Handle Events (Optional)

AdxSDK.showBanner({ containerId: 'ad-slot-1', size: '300x250', onAdLoaded: function(bid) { console.log('Ad loaded successfully!', bid); }, onAdFailed: function(error) { console.error('Ad failed to load', error); // Show fallback content } });

✨ SDK Features

  • ✅ Multiple Ad Formats: Banner, Native, Video
  • ✅ Automatic Tracking: Impressions, clicks tracked automatically
  • ✅ Viewability: Built-in viewability tracking
  • ✅ Responsive: Works on desktop, tablet, and mobile
  • ✅ User Privacy: Local storage for user ID, no cookies
  • ✅ OpenRTB Compatible: Industry-standard bid requests
  • ✅ Lightweight: ~12KB minified
  • ✅ No Dependencies: Pure vanilla JavaScript
